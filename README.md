# FRONTEND  
 - mở terminal
 - gõ lệnh `npm install`
 - gõ lệnh `npm start`
 - địa chỉ: `http://localhost:3000/`
 - tài khoản: `admin`
 - mật khẩu: `admin`
# BACKEND
#### cách 1
- ấn `nút tam giác(run)` ở góc trên phải màn hình (áp dụng cho IntelliJ IDEA và Visual Studio Code) (cách 1)
#### cách 2
- tải Java Development Kit (JDK)
- tải Maven
- lệnh chạy: `mvn spring-boot:run`
